#Tskupi
Tskupi is a web application that started off as a university project, but is currently being developed on a more personal scope. Tskupi makes it easier to play with friends and connect with people by providing interactive mini-games with what are essentially randomizing functions. As of right now, Tskupi has no back-end, and uses HTML, CSS, and JavaScript.

##Features
Although more features are to come, right now Tskupi offers:
-Coin flip
-Dice roll
-Random number generator
-Alias game
-Responsive layout
-User-friendly interface

##Privacy
Right now Tskupi does not process any personal data. The policy will be updated in case a login function is added.
